We needed to upload their routes, direct the Kamikaze to dive, and the drone needed to perform scans. It was a really comprehensive project, and it greatly increased my interest in UAVs. We needed to do the following with the ground control station. We would be able to observe both the drone and the kamikaze on the map. We would be able to upload their routes from the map, see their altitude, height, speed, and real-time status, and see if the engines were working. We would be able to see if they could return home in case of an emergency. They would be able to communicate with each other, and I had accomplished all of this. It was a really comprehensive project, perhaps even the most comprehensive project I had done up to that point.
